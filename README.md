# Eye-Follow

**Cursor Follow – Frontend Animation**

The **Cursor Follow** effect is an interactive frontend animation where an element smoothly follows the user's mouse cursor across the screen. It is built using **HTML, CSS, and JavaScript**, creating a modern and engaging user experience through real-time cursor tracking and smooth motion effects.

This project demonstrates key frontend concepts such as **DOM manipulation, mouse event handling, animation timing, CSS transitions, and responsive interaction design**. Cursor Follow animations are commonly used in creative portfolios, landing pages, and modern websites to enhance visual appeal and improve user engagement while showcasing advanced frontend development skills.
